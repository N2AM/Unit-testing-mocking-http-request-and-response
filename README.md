# Unit-testing-mocking-http-request-and-response

## Create project structure

-cache
-dist
-node_modules
-src - **mocksK** - images
-sass
-service - **tests** - .spec files - index.html

## Add parcel-bundler

`npm install -g parcel-bundler`

## Add sass

`npm install -D sass`

## Add Axios

`npm install axios`
